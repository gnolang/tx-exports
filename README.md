# tx exports

This repo serves the purpose of archiving data from testnets.

Currently, it contains the following:
- Raw transaction data from 
  - `test3.gno.land`
  - `test2.gno.land`
  - `test1.gno.land`
  - `staging.gno.land`
- Gno code extracted from raw transactions in the respective `extracted` folders.
